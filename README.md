# Harmoni Yoga Center 🌿

## Projektbeskrivning
Projektet **Harmoni Yoga Center** är en professionell webbplats utvecklad med **WordPress** och syftar till att presentera ett yogacenter på ett modernt och användarvänligt sätt.

Webbplatsen gör det möjligt att visa tjänster, kurser och teammedlemmar dynamiskt, samt ge besökare enkel tillgång till information om centret.

---

## Projektets mål
Målen med projektet är att:
- Skapa en professionell webbplats för ett yogacenter
- Tillämpa grundläggande webbutvecklingsprinciper
- Använda WordPress som CMS
- Förbättra användarupplevelsen (UX)
- Utveckla ett eget WordPress-tema
- Arbeta med dynamiskt innehåll

---

## Projektidé
Projektet bygger på idén att skapa en webbplats där administratören enkelt kan hantera:
- Yogatjänster
- Kurser och sessioner
- Instruktörer och personal  

Allt hanteras via WordPress administrationspanel utan behov av kodkunskap.

## Custom Post Types
### Services
Används för att visa:
- Yogatjänster
- Beskrivning av varje tjänst

### Team
Används för att visa:
- Instruktörer
- Bilder och presentationstexter

---

## Installation
1. Installera WordPress
2. Kopiera temat till:
 wp-content/themes/
3. Aktivera temat via adminpanelen
4. Skapa sidor och innehåll
5. Konfigurera menyer under:
 Appearance -> MENUS
